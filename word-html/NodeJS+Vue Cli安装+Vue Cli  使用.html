1.NodeJS安装（一个基于谷歌浏览器JS解释引擎V8开发的平台程序）

https://nodejs.org/en/download/

下载64位 .msi后，打开文件

2.安装Vue Cli（一个基于NodeJS开发的Vue客户端，方便编译、解释、打包Vue项目，集成了打包工具Webpack、JS高版本规范转低版本规范的转义工具babel）

2.1Cmd敲入命令行（npm是nodejs管理包的工具）

npm install -g @vue/cli

2.2安装完毕用Vue Cli可视化工具来构建、导入项目

vue ui

弹出了一个web页面

2.3 使用vue ui

2.3.1 创建项目

点按钮编辑下项目地址

输入项目名称

勾选 js转义插件babel，语法检eslint

*创建项目会导入nodejs包到项目的目录下过程有点卡，可以查看之前敲入的命令行vue ui的进度

上图是项目仪表盘（可以用来给当前项目部署插件，部署依赖，改配置，开发环境自动发布生成端口，生产环境编译导出项目到项目目录dist文件夹）

用WebStorm打开这个刚创建的项目就可以开发了

开发完成的时候编译项目了（点击项目仪表盘的任务，Vue Cli会识别.vue,然后自动打包）

编译成功，点击输出后，会有log输出地址给你打开查看，这个页面被加了WebSocket，任何开发代码改动都会自动编译，然后后台推送到这个网址
